# Antfetch
Another Rust fetch

## Building
Using `cargo` :
```bash
cargo build --release
./target/release/antfetch
```

If you don't have `cargo` :
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

The binary should appear in `/target/release`

## Downloading the binary

You can download the binary from the Releases tab
